#UPDATE Popularity in Movies
1) Create .env in directory with update_popularity.py

API_KEY=##
DB_USER=##
DB_PASSWORD=##
DB_HOST=##
DB_NAME=##

2) Create virtual enviroment and start it: "python -m venv div_env"
  run venv in linux: "source div_env/bin/activate"
  run venv in windows: "bin\Scripts\activate"

3) Install required packages > pip install -r requirements.txt

4) Edit range for Movies in try block

5) run python update_popularity.py
